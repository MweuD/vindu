What is Python?

Python refers to the Python programming language (with syntax rules for writing what is considered valid Python code) and the Python interpreter software that reads source code (written in the Python language) and per- forms its instructions. The Python interpreter is free to download from http://python.org/, and there are versions for Linux, OS X, and Windows. The name Python comes from the surreal British comedy group Monty Python, not from the snake. Python programmers are affectionately called Pythonistas, and both Monty Python and serpentine references usually pep- per Python tutorials and documentation.
